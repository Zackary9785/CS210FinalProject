CS155 Final Project

For this project I decided to implement an XOR gate with light switches for input, LED's for output and a sound functionality.



The truth table explains that there is an output when the inputs have different values.



The switches are both off, translates to 0 0 on truth table.



The first switch is off, second is on, translates to 0 1 on truth table.



The first switch is on, second is off, translates to 1 0 on truth table.



Both switches are on, translates to 1 1 on truth table
